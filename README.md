# CFD Visual Didático

Este projeto simula visualmente o escoamento aerodinâmico
em torno de um corpo sólido usando modelos matemáticos simplificados.

## O que é simulado
- Campo vetorial 2D de velocidade
- Linhas de corrente
- Mapa de velocidade em cores

## O que NÃO é
- Não resolve Navier-Stokes
- Não substitui CFD industrial

## Finalidade
- Ensino de aerodinâmica
- Portfólio técnico
- Visualização científica
